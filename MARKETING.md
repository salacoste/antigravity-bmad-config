## Recommended images (optional)

Create an `assets/` folder at repo root and add:

- `assets/banner.jpg` (README header banner)
- `assets/social-card.jpg` (GitHub Social Preview image)
- `assets/diagram.jpg` (optional “How it works” diagram)
- `assets/value.jpg` (optional value graphic for README)

### Where to use them

- README banner (top of `README.md`):
  - Add: `![Antigravity × BMad Method](assets/banner.jpg)`
- GitHub Social Preview:
  - GitHub repo → Settings → Social preview → Upload `assets/social-card.jpg`

 - Optional value graphic:

## Image prompts (Nano Banana)

You can generate all images in the same visual style (dark, minimal, “developer tool” aesthetic).

### 1) README banner (`assets/banner.jpg`)

**Size:** 1600×500 (or 2400×750 for retina)  
**Prompt:**
Minimal dark banner for a developer GitHub repository. Title text: “Antigravity × BMad Method”. Subtitle: “Slash-command AI agents for Agile/Scrum workflow: PRD → Architecture → Stories → Dev → QA”. Visual: subtle grid + thin orbit/antigravity motif + small icons for roles (PM, Architect, PO, Dev, QA). Modern, high contrast, clean typography, no mascots, no clutter. Plenty of negative space. **Size:** 1280×640

Small tagline: “Repeatable agentic Agile workflow”. Add a minimal flow line with 5 nodes labeled PRD, ARCH, STORY, DEV, QA. Professional, crisp, minimal, high readability at small sizes. PNG.

Labels: “/pm /architect /dev”, “workflows load personas + tasks”, “core-config.yaml resolves artifact paths”. Clean monospace labels, subtle lines, no busy decorations, high legibility. PNG.
